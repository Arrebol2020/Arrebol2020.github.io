AKAZE 和 ORB 平面跟踪

发表于2021-01-16|更新于2021-04-22|教程OpenCV
|字数总计:243|阅读时长:1分钟|阅读量:

介绍

在这个教程，我们将会比较 AKAZE 和 ORB 的局部特征，使用它们来寻找视频帧之间的匹配并跟踪物体的运动

算法如下：

  • 检测并描述第一帧的关键点，手动设置目标边界

  • 对于下一帧：

    • 检测并描述关键点
    • 使用 bruteforce matcher 进行匹配
    • 使用 RANSAC 估计 homography transformation
    • 对所有的匹配进行过滤 inliers
    • 对边界框使用 homography transformation 来查找对象

    • 绘制边界款和 inliers ，计算 inlier ratio 作为评估指标

数据

为了进行跟踪，我们需要在第一帧上显示视频和对象的位置

视频和数据下载地址：https://docs.google.com/file/d/0B72G7D4snftJandBb0taLVJHMFk

执行代码前，必须制定输入(摄像机 id 或视频文件)。随后用鼠标选择一个边框，按任意键开始跟踪

./planar_tracking blais.mp4
