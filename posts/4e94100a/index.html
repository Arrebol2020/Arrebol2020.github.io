吴恩达《机器学习》5

发表于2020-11-05|更新于2021-04-23|学习课程吴恩达《机器学习》
|字数总计:1.4k|阅读时长:4分钟|阅读量:

决定如何优化

当训练结果模型应用于新的数据集表现的效果很差时，可以尝试以下方法：

  • 分析出模型是欠拟合，即具有高偏差
    • 添加更多的特征
    • 增加更复杂的多项式特征
    • 减小 λ (正则化参数)
  • 分析出模型是过拟合，即具有高方差
    • 获取更多的训练样本
    • 尝试更少的特征
    • 增大 λ (正则化参数)

机器学习诊断法(machine leanring diagnostic)

这是一种测试法，通过执行这种测试，能够了解算法在哪里出了问题，也能够告诉你要想改进一种算法的效果，什么样的尝试才是有意义的

数据集的划分

如果用训练集来拟合参数 Θ 时，那么拟合后的模型在训练集的效果是不能预测出假设函数对于新样本的泛化能力的。这是因为这些参数能够很好地你和训练集，因此有可能在训练集上表现的很好，但对于其他的新样本来说就不一定那么好了。

训练集划分的方式：

  • 划分为训练集和测试集(大部分情况下)
  • 划分为训练集、交叉验证集和测试集(对于需要选择参数，训练多个模型，从中选取最好的模型)

诊断偏差和方差

  • 对于高偏差(欠拟合)的情况下，可以发现交叉验证集和训练误差都很大
  • 对于高方差(过拟合)的情况下，可以发现训练集的误差很小，交叉验证集的误差很大

正则化和偏差与方差

假如我们要对上图那样一个高阶的多项式进行拟合，为了防止过拟合，添加了正则化项

  • 第一种情况，λ 非常大，在这种情况下，所有这些参数包括(θ1，θ2，…，θn)大部分都接近于 0 ，并且这个假设函数将等于或近似于 θ0。因此最终将得到的假设函数近似是一条水平的直线，这个假设函数处于高偏差，对于数据欠拟合
  • 第二种情况，λ 大小适中，我们能到到一组对数据拟合比较合理的 θ 参数值
  • 第三种情况，λ 非常小，在这种情况下，要拟合一个高阶多项式，通常会出现过拟合的情况

学习曲线(learning curve)

随着训练集容量的增大，平均训练误差在增大。在训练样本很少的时候，对每一个训练样本都能很容易地拟合的很好，所以训练误差将会很小，当 m 值逐渐增大，那么想对每一个训练样本都拟合到很好，就显得愈发困难了，所以训练集的误差就会越来越大

交叉验证集误差，当训练集很小的时候，泛化程度不会很好，即不能很好地适应新样本，误差较大，只有当使用一个更大的训练集时，才有可能得到一个能够更好拟合数据的假设函数，误差将会减小，因为使用的数据越多，越能获得更好地泛化表现

高偏差的学习曲线

如果一个算法存在高偏差，随着我们增加训练样本，会发现交叉验证误差不会明显的下降了，即使用更多的训练样本对于改进该算法的性能并没有任何作用

高方差的学习曲线

如果一个算法存在高方差，如果继续增大训练样本的数量，交叉验证集误差将会一直下降，所以在高方差的情形中，使用更多的训练集数据对改进算法是有帮助的

误差分析

开发机器学习应用的过程：

  • 通过一个简单的算法来快速地实现它，并且使用交叉验证集数据来验证测试
  • 绘制相应学习曲线以及检验误差来找出当前使用的算法是否存在高偏差或高方差的问题或一些其他问题，在做出这些分析之后再来决定是否使用更多的数据或特征等等
  • 误差分析：通过查看哪些被错误分类的样本，有什么共同的特征和规律，这样可能可以设计出新特征来提升模型的性能

阳性与阴性以及正类别与父类别

  • 真正例(TP)：将正类别样本正确地预测为正类别
  • 真负例(TN)：将负类别样本正确地预测为负类别
  • 假正例(FP)：将负类别样本错误地预测为正类别
  • 假负例(FN)：将正类别样本错误地预测为负类别

精确率(precision)

在被识别位正类别的样本中，确实为正类别(TP)的比例是多少

Precision = TP / (TP+FP)

召回率(recall)

在所有正类别样本中，被正确识别为正类别的比例是多少

Recall = TP / (TP+FN)

精确率和召回率的trade-off

要全面评估模型的有效性，必须同时检查精确率和召回率。但是提高精确率通常会降低召回率

F1 Score

精确率和召回率是相互影响的，精确率升高则召回率下降，召回率升高则精确率下降，如果需要兼顾二者，就需要精确率，召回率的结合

F1-Score = 2 / (1/Precision + 1/Recall)

Reference
