寻找两个正序数组的中位数

发表于2021-01-27|更新于2021-04-20|代码相关LeetCode
|字数总计:508|阅读时长:2分钟|阅读量:

题目链接

题目链接：https://leetcode-cn.com/problems/median-of-two-sorted-arrays/

题目描述

给定两个大小为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的中位数。

进阶：你能设计一个时间复杂度为 O(log (m+n)) 的算法解决此问题吗？

示例 1

输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2

示例 2

输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5

示例 3

输入：nums1 = [0,0], nums2 = [0,0]
输出：0.00000

示例 4

输入：nums1 = [], nums2 = [1]
输出：1.00000

示例 5

输入：nums1 = [2], nums2 = []
输出：2.00000

提示

  • nums1.length == m
  • nums2.length == n
  • 0 <= m <= 1000
  • 0 <= n <= 1000
  • 1 <= m + n <= 2000
  • 106 <= nums1[i], nums2[i] <= 106

本人解法

思路

因为两个输入的数组都是有序的。首先将两个数组合并，合并之后对数组进行排序。排序后根据题目要求输出中位数。这种解法，简单、粗暴但是很费空间
