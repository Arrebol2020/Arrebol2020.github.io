李宏毅《机器学习》1 Introduction

发表于2020-11-21|更新于2021-04-20|学习课程李宏毅《机器学习》
|字数总计:1.1k|阅读时长:3分钟|阅读量:

Supervised Learning(监督学习)

我们使用监督学习这样一种技术时，我们需要告诉机器， function 的输入和输出分别是什么，而这种输出通常是通过人工的方式标注出来，因此成为人工标注的 label ，它的缺点是需要大量的人工

Regression(回归)

回归是机器学习的一个任务，其通过回归找到一个函数，该函数的输出是一个标量数值。比如 PM2.5 的预测，给机器的训练集是过去的 PM2.5 资料，而输出的结果是对未来 PM2.5 的预测数值

Classification(分类)

回归和分类的区别是：在回归问题中，输出的是标量，而在分类中又分为如下两类：

  • Binary Classification(二分类)：在二分类中，我们要机器输出的是 0 or 1 ，比如 G-mail 的垃圾邮件过滤器，输入的是邮件，输出是该邮件是否属于垃圾邮件
  • Multi-class classification(多元分类)：在多元分类中，机器要做的是选择题，等于给他数个选项，每一个选项就是一个类别，它要从数个类别里面选择正确的类别，比如新闻文章分类，输入是一则新闻，输出是这个新闻属于哪一个类别

Model (function set)选择模型

在解决机器学习任务的过程中，第一步就是要选择 function 的集合，选择不同的 function 集合，会得到不同的结果；而选不同的 function 集合就是选择不同的 model ，model 又分为很多种：

  • Linear Model(线性模型)：最简单的模型
  • Non-linear Model(非线性模型)：最常用的模型，其包括：
    • Deep learning(深度学习)
    • SVM
    • Decision tree(决策树)
    • K-NN

Semi-supervised Learning(半监督学习)

例如：如果想要做一个区分猫和狗的分类，手头上有少量的标签数据，它们标注了图片上哪只是猫哪只是狗；同时又有大量的没有标签的数据，它们仅仅只有猫和狗的图片，但没有标注去告诉机器哪只是猫哪只是狗。在半监督学习的技术里面，这些没有标签的数据，对机器学习也是有帮助的

Transfer Learning(迁移学习)

例如：如果想要做一个区分猫和狗的分类，但是我们现在有大量的不相干的数据(不是猫和狗的图片，而是一些其他不相干的图片)标签。迁移学习要解决的问题是，这一堆不相干的数据可以对结果带来什么样的帮助

Unsupervised Learning(无监督学习)

与监督学习不同，无监督学习希望机器自学，在完全没有任何标签的情况下，机器到底能学到什么样的知识。举例来说，如果我们给机器看大量的文章，机器看过大量的文章之后，它到底能够学到什么事情？它能不能学会每个词汇的意思？

学会每个词汇的意思可以理解为：我们要找一个 function ，然后把一个词汇丢进去，机器要输出告诉你说这个词汇是什么意思，也许他用一个向量来表示这个词汇的不同的特性

Structured Learning(结构化学习)

在结构化学习里，机器的输出是一个有结构性的东西。例如：在机器的输入是一个声音信号，输出是一个句子；句子是由许多词汇拼凑而成，它是一个有结构性的对象

Reinforcement Learning(强化学习)

在强化学习中，我们没有告诉机器正确的答案是什么，机器最终得到的只有一个分数，这个分数用于评估机器做的好还是不好，然后机器会自己进行学习，不断的进行摸索学习。比如训练一个聊天机器人，让它跟客人直接对话；如果客人勃然大怒把电话挂掉了，那机器就学到一件事情，刚才做错了，它不知道自己哪里做错了，必须自己回去反省检讨到底要如何改进，比如一开始不应该打招呼吗？还是中间不能骂脏话之类的
