论文笔记 Revisiting Training Strategies and Generalizaion Performance

发表于2020-11-24|更新于2021-04-22|论文论文笔记
|字数总计:3.6k|阅读时长:11分钟|阅读量:

简介

由于目前 DML 研究方向多样化，例如：架构、目标函数 和额外的训练任务，这导致了对他们的研究的结果进行公平的比较变得越来越困难，此外，有些作者并没有公布技术的细节，这使得研究人员进行再现实验结果变得更加困难。

所以作者回顾了目前最广泛使用的 DML 目标函数，且对关键参数和 mini-batch抽样过程进行了研究。通过研究发现 DML 目标优化饱和度已经很高了。基于进一步的分析，作者发现嵌入空间密度和压缩与 DML 模型的泛化性能之间存在着关联性。

下图是在标准基准数据集，在相同的训练条件下，对主流 DML 方法的比较结果

在此基础上，作者将分析拓展到：

  • 影响 DML 的因素之一的数据采样过程。以往的研究只是在 ranking-based 目标的 triplet mining strategies 的特定背景下研究该过程。而作者检测了 model-agnostic 情形下抽样信息的 mini-batch
  • 通过分析 DML 模型学习到的嵌入空间的结构来评价其泛化能力。虽然目前不能可靠地将典型的目标概念，如 large inter-class margins 和 intra-class variance 和泛化性联系起来，作者发现压缩学习的 representations 与 泛化能力有很强的联系
  • 基于上述发现，作者提出了一个简单而有效的训练正则化方法，该方法能可靠地提高 ranking-based DML 的模型，在各种 benchmark 的数据集上的性能

学习视觉相似性对与很多的视觉任务十分重要，例如：图像聚类、人脸识别和图像检索。测量相似度需要学习一个嵌入空间，该空间通过定义的距离度量，从而合理地反映图像的相似度。对于这样的任务，最常使用的算法之一是 DML ，它使用 深度神经网络 来学习 preserving embedding

作者主要贡献如下：

  • 作者对于主流的 DML 目标函数、训练策略、数据采样的影响和模型设计选择进行了详细的分析，以建立一个 standard benchmark
  • 通过分析 DML 泛化与嵌入空间压缩的相关性、 inter-class margins 和 intro-class variance ，作者对于 DML 性能 提出了新的见解
  • 基于上述结论，作者提出了一种简单的正则化嵌入空间压缩的技术，它能可靠地提高 ranking-based DML 的模型的泛化性能

Deep Metric Learning

DML 在图像检索和人脸识别应用中，变得越来越重要。许多方法都使用了 ranking-based objectives 基于 tuple 样本，例如 pairs、triplets或者更高的 variants (如 N-Pairs、lifted structure losses、NCA-based criteria)。此外 classifaction-based methods 的方法已经被证明能够有效的学习 distance preserving 嵌入空间。

为了解决 tuple-based 方法计算复杂度高的问题，引入了许多采样策略。

  • proxy-based 方法通过使用少数 virtual representatives 来近似类分布
  • Sanakoyeu et al. 采用将数据和嵌入空间分离随后合并的分而治之的策略
  • Opitz et al. 提出部署一个 ensemble 的特殊学习器的策略
  • Roth et al. 提出 feature mining 或自监督学习与 DML 结合的策略
  • Lin et al. 提出生成人工样本来有效的增大训练集，从而能够学习到更多复杂的 ranking relations 的策略

Generalization in DML

根据由强大的实验支持所覆盖的压缩，在分类的上下文中将压缩连接到 representations 的扁平化。作者应用这个概念来分析 DML 的泛化，发现强压缩实际上会损害 DML 的泛化能力。目前对于 DML 泛化能力的研究主要集中于线性或基于核的距离度量的鲁棒性和检查泛化误差的边界。与此不同，作者研究了泛化和学习的嵌入空间结构特征之间的相关性

Mini-batch selection

通过对 mini-batch 的研究发现，mini-batch 可以提高分类任务优化方法的收敛性，或者利用 core-set seletion 构建信息的 mini-batch 用于优化生成的模型

上述方法的大多数都是使用基本的目标函数进行训练的，并且进一步依赖于作者研究讨论的训练参数

The objective function

  • Ranking-based：是目前最流行的目标函数。其中有：pairs、triplets 或者 larger sets of datapoints 。该方法将学习 φθ 定义为一个使同易类别的 锚点 xa 和 positive xp (即 ya=yp ) 之间的距离 dφ(xa, xp) 最小，使不同类别标签的负样本 xn 之间的距离 dφ(xa, xn) 的距离最大 的 ordering task

  • Proxy-based：该方法将整个类的分布用一个更好的 learned representatives 来表示。通过考虑 class representatives 来计算训练损失，可以直接将单个样本与整个类进行比较。此外，该方法有助于缓解 ranking-based 损失函数中遇到的 tuple mining 问题

Data Sampling

Mining informative mini-batches B 过程不依赖于具体的训练目标，但是目前在 DML 的研究中经常被忽视。

  • label sampler：它是只遵循基于标签的选择规则的抽样启发式方法

    • SPC-2/4/8：给定 batch-size b ，随机选择 b/n 个唯一的类，从中随机选择 n 个样本
    • SPC-R：我们从数据集中随机选择 b-1 个样本，并选择最后一个样本与 b-1 个样本中的一个具有相同的标签，以确保可以从 b 中发掘出一个 triplet 。因此，可以有效地在 mini-batches 中改变 unique class 的数量

  • embedded sampler：它对数据嵌入本身进行操作，以创建不同数据统计信息 batches B。增加 batch-size b 已被证实有益于稳定优化，因为有更大的数据多样性和更多的训练信息。但是 batch-size 受限于内存。为了虚拟最大化数据的多样性，在 embedding φ 下，通过匹配 B 和 B 的统计数据，将一组大样本的信息 B 和 b = |B| > b 的内容提取到一个 mini-batch B 中。为了避免计算开销，作者从一个不断更新的内嵌训练样本的内存库 M 中抽取 B 。 M 是根据训练的 batch B 来迭代更新其元素。使用 M ，用于 mine mini-batch，首先从 M 中随机抽样 B，b*=1024 ，之后通过使用以下标准之一找到一个 mini-batch B 用于匹配它的数据统计：

    • Matching of distance distributions(DDM)：其目标时保持 B 的距离分布。随机 m 个 mini-batches ，并选择 B 和 B 的标准化距离直方图中 Wasserstein distance 最小的 batch B
    • FRD-Score Matching(FRD)：与 GAN evaluation setting 相似，计算 B 和 B* 之间的 frechet distance 来测量它们之间的相似度

Training parameters，regularization and architecture

  • Architechtures：主流网络架构为：GoogLeNet、Inception-BN、ResNet50(可以冻结 batch-normalization layers，以改进跨不同批处理大小的收敛性和稳定性)，但是通常只有使用单一架构的结果，这损害了公平比较
  • Weight Decay：通常使用 weight decay / L2 正则化进行网络优化。在 DML 中，尤其是在小数据集上，需要仔细调整，来最大化泛化性能
  • Embedding dimensionality：选择一个 embedding space 的维数 D 影响泛化能力。虽然每个架构通常在 DML 中使用一个单独的、标准化的维度 D，但是最近的研究报告进行了适当的修改，但并没有报告适当的基线
  • Data Preprocessing：数据预处理图像通常显著影响着学习特征和模型正则化。因此，由于最近的方法在其应用的增强方案上有所不同，结果不一定具有可比性
  • Batchsize：batchsize 决定了梯度更新到网络的性质。对于大的 batchsize，可以拥有许多类，从而更好地近似训练分布。然而并没有将其作为变异的影响因数加以考虑

实验所使用的数据集

  • CUB200-2011
  • CARS196
  • Stanford Online Products (SOP)

Experimental Protocol

  • 使用 ResNet50 架构(因为它在最近的 SOTA 方法的成功应用)，冻结 batch-normalization layers 和 embedding dim 128
  • 将图像随机 resize 和 crop 为 224 × 224 用于训练，并将 center crop 成相同大用于评估
  • 在训练过程中，采用 horizontal flipping(p = 0.5)
  • 使用 Adam 优化，学习率固定为 10^-5，无 learning rate sheduling， weight decay 为 4 × 10^-4
  • 使用 PyTorch 构建模型，Nvidia itan X, V100 and T4 GPUs 12 GB
  • 每次训练在 CUB200-2011/CARS196 执行 150 个 epochs，Stanford Online Products 执行 100 epochs
  • 使用 SPC-2 策略进行 batch sampling
  • 每个结果在多个 seeds 上求平均，以避免基于 seed 的性能波动
  • 评估指标：Recall、Normalized Mutual Infomation、F1 score 和 召回率测量的平均精度(mAP)
  • 对于每个损失，每个度量标准都按照评估值范围内的最大值进行标准化，这使得跨多有度量的性能汇总成为可能，其中差异对应于相对的改进

Studying DML parameters and architectures

下图为基于上述设置，固定其他参数，分析的结果：

  • batchsize： 通常增加 batchsize 会改善结果。在不同的标准中获益不同。从图中可知，特别是在 SOP 数据集上具有很高的相关性
  • weight decay：从图中观察到损失和依赖于数据集的行为，达到了 5% 的相对性能的变化
  • embedding dim：增加输入图像的大小，会导致很大的性能差异。无论目标和数据集，较大的图像都能够提供更好额性能。
  • preprocessing：随机调整大小的裁剪，通常比基本的调整大小和裁剪更有效
  • architecture：使用不同的 D 会得到不同的结果，对于 R50 来说，似乎在 D = 512 处收敛。但是 R50 的 D 通常固定为 128。

Batch sampling impacts DML training

作者研究返现 batch sampling process 确实会影响 DML 训练的平均表现，差异高达 1.5% 。且观察到 SPC-2 和 FRD 采样器表现非常好。 DML 受益于 mini-batch 的数据多样性，独立于所选择的培训目标

Comparing DML models

对于 ranking-based model ，作者采用了 distance-based tuple mining (其被证明是最有效的)，同时引入了 random 、semihard sampling 和 a soft version of hard sampling 用于 tuple mining 研究使用经典的 triplet loss。

loss-specific 超参数是通过小的交叉验证网格搜索来确定的，这些网格搜索围绕最初的提出值进行调整，以适应训练设置。

作者注意到，在 CUB200-2011 和 CARS196 上，由于数据分布上的强烈差异，与 SOP 相比，不同方法之间的性能饱和度更高。ranking-based 目标的 representatives 总体上略优于基于分类/NCE 的对应目标。通常 margin loss 和 multisimilarity 提供了跨数据集的最佳性能

经过仔细训练的 baseline models 能够超过使用相当强大架构的最先进的方法。

Generalization in Deep Metric Learning

最近的研究将压缩和 class-conditioned flattening of representation 联系在一起，通过对数据表示的奇异值分解(Singular Value Decomposition, SVD)得到的奇异值的增加衰减来表示。因此，类表示占据了更紧凑的空间，从而减少了具有显著差异的方向的数量。随后对最具鉴别性方向的关注被证明对具有识别训练和测试分布的经典分类场景是有益的。然而，这过分地抛弃了能够捕获训练分布之外的数据特征的特征。因此，作者假设，实际上保留相当多的显著方差方向(DoV)是学习一个良好的推广嵌入函数的关键

DML 的泛化与学习表示的 SV 谱衰减呈强你相关，与嵌入空间密度呈弱相关。这表明，在训练和测试分布之间有较大变化的情况下，过度的特征压缩会损害表征学习，但更大的嵌入空间的密度可能有益于表征学习

ρ-regularization for improved generalization

作者利用他们的发现，通过对抗表示的压缩，提出了一种简单的基于排序的正则化方法。

  • 通过概率 p_switch 将负样本 xn 与正样本 xp 交换，随机改变 tuples 。这样就将同一类的样本分开了，使得 DML 模型能够捕获额外的非标签，区分特征，同时抑制由强区分训练引起的压缩。尽管训练分布具有分离所有测试类所需要的特征，但这些特征被强区分性训练信号所忽略。加快压缩通过衰减谱衰减 ρ(Φ) 使模型来捕捉更多的信息，因此在没见过的测试类中表现出很强的泛化能力。ρ-regularization 降低了 spectral decay ，从而提高了 recall 性能

作者研究发现，高度压缩的表示忽略了捕获传输到位置测试分布的数据特征的有用特性。为此提出了一种简单的基于排序的方法来规范嵌入空间的压缩，从而提高了所有基准数据集的性能
