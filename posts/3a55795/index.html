Deep Learning

上图中，每一列表示一个 layer ，每个 layer 里面的每个球都代表一个 neuron

  • layer 和 layer 之间 neuron 是两两互相连接的， layer 1 的 neuron output 会连接给 layer 2 的每一个
    neuron 作为 input
  • 对整个 neural network 来说，它需要一个 input ，这个 input 就是一个 feature 的 vector ，而对 layer
    1 的每一个 neuron 来说，它的 input 就是 input layer 的每一个 dimension
  • 最后那个 layer L ，由于它后面没有接其它东西了，所以它的 output 就是整个 network 的 output
  • 这里每一个layer都是有名字的
    • input 的地方，叫做 input layer ，输入层(严格来说 input layer 其实不是一个 layer ，它跟其他 layer 不一样，不是由 neuron 所组成的)
    • output 的地方，叫做 output layer ，输出层
    • 其余的地方，叫做 hidden layer ，隐藏层
  • 每一个 neuron 里面的 sigmoid function ，在 Deep Learning 中被称为 activation function (激励函数)，事实上它不见得一定是 sigmoid function ，还可以是其他 function (sigmoid function 是从 Logistic Regression 迁移过来的，现在已经较少在 Deep learning 里使用了)
  • 有很多层 layers 的 neural network ，被称为 DNN (Deep Neural Network)

上图中，层与层之间，所有的 neuron 都是两两连接，所以叫做 Fully connected 的 network ；因为传递的方向是 layer 1->2->3 ，由后往前传，所以它叫做 Feedforward network

Hidden layers 部分可以看做是一个特征提取器( feature extractor )，这个特征提取器取代了之前手动做特征工程( feature enginerring ) 和 特征转化( feature transformation ) 这些事情，经过特征提取器得到的 x1，x2，…，xk 就可以看做是一组新的 feature

设计网络结构

网络结构( network structure )设计是很重要的一件事，之前在做 logistic regression 或 linear regression 的时候，对于模型的结构是没什么好设计的，但是对 neural network 来说，对隐藏层的设计十分重要(即有多少隐藏层，每一层有多少个 neuron )，它决定了 function set 。

设计网络结构和特征工程

  • 在机器学习中，需要手工提取特征。
  • 在深度学习中，需要手工设计网路结构
