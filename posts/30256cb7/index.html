数据压缩

数据压缩可以让学习算法运行得更快，减小内存使用，减小磁盘空间需求和用于可视化分析数据

主成分分析(principal component analysis, PCA)

PCA：它会试图找一个投影平面对数据进行投影，使得能最小化这个距离(使用 PCA 之前需要先进行均值归一化和特征缩放，使得特征量其均值为 0 ，并且其数值在可比较的范围内)。若想将数据从二维降到一维，我们要试着找一个向量，要找一个数据投影后能够最小化投影误差的方向

在通常情况下，我们会有 N 维的数据，并且我们想其降到 K 维，这种情况下，我们不只是想找单个向量来对数据进行投影，而是想寻找 K 个方向来对数据进行投影，来最小化投影误差

PCA 与 线性回归

PCA 不是线性回归，它们实际上是两种不同的算法。

  • 线性回归要锁的是用所有的 x 值来预测 y值，当我们处理线性回归时，用做图表示我们要做的，当给定某个输入特征量 x 时来预测出某变量 y 的值，因此在线性回归中，我们要做的是拟合一条直线，来最小化点和直线之间的平方误差，所以我们要最小化的是这些蓝线之和的平方
  • 在处理 PCA 中，它要做的是试图最小化这些蓝色直线的长度，它们是最短的正交距离。在 PCA 中，没有区别对待数据，没有什么特殊的变量 y 是我们要预测的，相反，我们的一系列特征 x1，x2，…，xn 都是同等对待，因此它们没有一个是特殊的

PCA 数据的预处理

给定一个数据集，一定要做的是执行均值标准化，如果不同的特征有非常不相同的缩放，要进行特征缩放，使它们处于相对的价值范围。

PCA 使用限制

当在运行 PCA 时，仅仅在训练集中的数据上运行，不能用在交叉验证集和测试集数据，定义了 x 到 z 的映射后，才可以应用这个映射到交叉验证集和测试集

PCA 最好不要用于防止过拟合。在执行 PCA 之前，首先考虑使用最原始的数据 x ，只有这么做不能够达到目的的情况下才考虑使用 PCA 。因此在使用 PCA之前，与其考虑减少数据维度步入考虑抛弃 PCA 这一步，仅考虑在原始数据上训练学习算法，只有在学习算法运行太慢或者需要的内存或者硬盘空间太大，才使用 PCA
